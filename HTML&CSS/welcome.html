What is the best way to learn HTML

Start by working through www.HTMLdog.com . Do not use w3schools, it has erroneous information. Look at www.w3fools.com, to see why. Next read Designing with Web Standards by Jeffrey Zeldman. When you understand HTML and CSS, then explore other web development options, e.g. JavaScript.

  • Build your own web site
  • Find a website you like, copy their source and hack it until you like it
